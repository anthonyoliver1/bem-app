O que é o BEM ?

BEM é uma metodologia, um padrão de nomenclatura que podemos usar para facilitar o desenvolvimento e manter nosso projeto simples e organizado.

Essa metodologia é aplicada em classes CSS dos elementos HTML. A sigla BEM significa Block Element Modifier (Bloco Elemento Modificador) - esses são os 3 pilares base da metodologia aplicada pelo BEM e são as categorias onde vamos dividir os elementos.

Análise o código abaixo: 

        {{codeExemple1}}

Podemos destacar no código acima alguns pontos:

  • .list é um bloco;
  • .list__item, .list__title, .list__author e list__text são elementos;

Vamos entender:

  • Nossa primeira classe sempre será o bloco: .list
  • Os elementos usam 2 underlines (_ _) após o nome do bloco: .list__item, .list__title, .list__author e list__text
  • Para os modificadores, usamos 2 traços (--) no bloco ou elemento: list__item--highlight, .list__author--active
